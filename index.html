█ BBS Theme Tool █

About

The BBS Theme Tool is a Python application for managing themes in the Minecraft BBS mod. It allows you to import and export theme configurations, modify colors, and manage assets.

Prerequisites

Before running the BBS Theme Tool, ensure you have:

  • Python 3.x - Download from python.org
  • Pillow (PIL) - Install using pip
pip install pillow

How to Run

To start the BBS Theme Tool:

python theme.py
✓ The application will open in a new window with the graphical interface

Features

  • Theme Import: Apply themes to BBS configuration
  • Theme Export: Create new theme packages
  • Config Path Browser: Easy directory selection
  • Theme Preview: See colors and backgrounds before applying
  • Search Functionality: Filter available themes
  • Asset Management: Handle background and icon files

Usage

  1. Click "Browse" to select your BBS config directory (should contain settings/bbs.json)
  2. Choose either "Import" or "Export" mode
  3. For Import: Select a theme from the list and click "Execute"
  4. For Export: Enter a name and choose format (ZIP or Folder)
  5. Use "Preview" to see theme colors before applying
Note: This is a Python desktop application. The web interface you're viewing is for documentation purposes only.
